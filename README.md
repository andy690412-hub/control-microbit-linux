# 마이크로비트 기반 이족보행 로봇 원격 제어

*1. 개요*

마이크로비트와 서보모터를 활용하여 기계식 캠(Cam) 구조의 이족보행 로봇을 구현한 프로젝트입니다. 파이썬을 기반으로 리눅스 PC에서 시리얼 통신을 통해 로봇의 동작(전진/정지)을 실시간으로 제어합니다.

*2. 기능*

원격 제어 시스템: 리눅스 환경에서 파이썬 시리얼 통신(pyserial)을 통해 마이크로비트에 제어 신호(1: 전진, 0: 정지)를 전송합니다.
물리적 모션 제어: microbit.pin 모듈을 활용하여 서보모터의 PWM(아날로그) 신호를 조절, 이족보행 메커니즘 반복 동작을 수행합니다.
상태 시각화: 마이크로비트 전면 LED를 통해 현재 로봇의 동작 상태(전진: 화살표, 정지: 수면)를 직관적으로 출력합니다.

*3. 세부사항*

언어: Python (bash), MicroPython (로봇 제어 코드)
통신: Serial Communication (UART, 115200 bps)
하드웨어: BBC Micro:bit, 서보모터(Servo Motor), 캠 구조 기반 이족보행 키트
